## 极简任务管理-飞书多维表格

多人协作，用飞书多维表格还是不错的，主要优势是:
1. 如果公司使用了飞书，人员接入不需要了
2. 任务总体数量比Bug少，所以性能够用
3. 操作和配置方便，同时支持多个视图
4. 可以支持机器人和流程自动化，当任务状态发生变化，可以自动推消息，比如提测
5. 人不算太多，都是免费的

## 任务管理方式

任务管理主要就是：
1. 分组/分任务/分负责人
2. 定时间点
3. 可以看到进行进度，追踪总体进度
4. 变化可以及时通知
